# fit

a flutter Syntactic sugar

## Usage

```shell
flutter pub add fit
```

## Example
```dart

class MyWidget extends StatelessWidget {
  const MyWidget({Key? key}) : super(key: key);
  @override
  Widget build(BuildContext context) {
    return Text("123")
        .fontSize(20)
        .color(Colors.red)
        .padding(all: 10);
  }
}

```
